# Short blurbs — Navidrome Discord & Lemmy

## Navidrome Discord (#showcase / #plugins)
> Built a little companion for Navidrome: you type a vibe like *"rainy evening reading a book"* and
> it turns it into readable audio-feature ranges (relaxed/energy/bpm…), matches your library's
> essentia analysis, and makes a normal playlist. Comes with an optional plugin for scheduled
> "Vibe of the Day" lists. Local-first (Ollama by default), GPL-3.0, early/solo — feedback very
> welcome 🙏 https://github.com/craiglush/navidrome-vibe

## Lemmy — c/selfhosted (or c/opensource)
**Title:** Vibe Playlists — describe a mood, get a playlist (self-hosted, for Navidrome/Subsonic)

> Author here. Open-source companion for Navidrome: type a vibe and it creates a playlist by
> mapping your words to human-readable audio-feature ranges over essentia analysis (so you can see
> *why* each track was picked), then writes it back via the Subsonic API so it shows up in every
> client. Runs in Docker, local-first (your own Ollama; cheap hosted model optional, no GPU
> needed), GPL-3.0. Early and solo — would love feedback from other setups.
> https://github.com/craiglush/navidrome-vibe

> Tip: don't post the Discord and Lemmy versions within minutes of each other; space them out, and
> reply to whatever comes back.
